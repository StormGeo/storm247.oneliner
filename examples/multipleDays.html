Be warned that this example blocks page rendering until script tag is loaded. See other async options for better performance. However this is perfect solution if you bundle widget script with your own scripts.

{dayOfWeek(2)} {Days[2].Day.TemperatureMin.round}° / {Days[2].Day.TemperatureMax.round}°
